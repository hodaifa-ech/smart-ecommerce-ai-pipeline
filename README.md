# Getting Started 
```
git clone https://github.com/hodaifa-ech/smart-ecommerce-ai-pipeline
cd smart-ecommerce-ai-pipeline
pip install -r requirement.txt 
streamlit run main.py
```

**requirements:**
```
export GROQ_API_KEY="api_key"                                    
```
